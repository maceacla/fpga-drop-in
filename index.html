Hands-On FPGA Learning
for Every EE Student

At Carleton University’s Department of Electronics, we prioritize hands-on learning to prepare undergraduate electrical engineering students for their careers. Field-Programmable Gate Arrays (FPGAs) are powerful tools used in fields like signal processing, communications, and embedded systems, but many students are unfamiliar with them when they begin their studies. Our FPGA Drop-In Center addresses this gap by introducing students to FPGAs and providing the resources to explore and master them.

Serving approximately 1,950 students each year, the drop-in center is a welcoming space where students of all skill levels can work with FPGA development boards. Through practical labs, guided tutorials, and peer-supported workshops, students learn to design and test circuits, gaining confidence with FPGAs. The center supports four core courses, offering projects like building logic circuits or programming embedded systems. This approach ensures every student, regardless of prior experience, can engage with FPGAs and develop valuable skills for industry and research.

By making FPGAs accessible and providing clear, structured support, the drop-in center helps students move from curiosity to competence. It’s a place where learning by doing builds a strong foundation for electrical engineering success.

Why Learn with FPGA?

FPGAs are integral to modern applications like AI, signal processing, and embedded systems. They offer flexibility and performance, making them essential tools in today’s engineering landscape.

Hands-on experience with FPGAs enhances employability and prepares you for industry demands. At Carleton, our students gain practical skills that set them apart in the job market.

The Drop-In Center provides access to real hardware, fostering practical skills over simulations. This approach ensures you’re ready to tackle real-world engineering challenges.

  • Real-world applications in AI and DSP.
  • Enhanced career prospects.
  • Practical, hands-on learning.

Sample FPGA Projects

1st Year - Introductory

Focus: Logic, State Machines, Basic I/O

Binary Clock

Learn timekeeping, counters, and seven-segment displays.

Reaction Timer Game

Combines switches, LEDs, and basic timing logic.

Simple Calculator (4-bit inputs)

Finite state machine design, BCD, and combinational logic practice.

Morse Code Encoder/Decoder

Bit encoding, shift registers, and output control.

LED Music Visualizer

Use basic amplitude sampling with hardcoded audio input for LED pattern output.

2nd Year - Intermediate

Focus: Embedded Integration, Control, IO Devices

UART Terminal Interface

Design a UART controller for text-based communication with a PC terminal.

Temperature Monitor (I2C)

Teaches I2C protocol implementation and digital interfacing with a sensor.

Basic VGA Controller

Introduces video timing, frame buffers (e.g., draw shapes or grids).

Digital Lock with Keypad

FSM with user input, password storage, and error handling on seven-segment display.

Pong Game on VGA

Builds on VGA controller with basic collision logic and simple game loop.

3rd Year - Applied Systems

Focus: Integration, Signal Processing, Real-Time

PWM Motor Controller

Real-world motor control, feedback loops, and analog interfacing (via ADC).

Audio Equalizer (Basic DSP)

FPGA processes audio in real-time, with adjustable gain per band.

FPGA Logic Analyzer

Capture and display simple digital signal waveforms triggered by edge detection.

IR Remote Decoder

Teaches signal timing, decoding (e.g., NEC protocol), and control output.

Digital Synthesizer

Generates chiptune-style waveforms from user input, DAC output via audio jack.

4th Year - Advanced

Focus: Embedded Systems, Computation, Mixed-Signal

RISC-V Soft Processor

Build and test a simple CPU implementation, optionally with peripherals.

Face Detection Accelerator

Combines OpenCV pre-processing with FPGA for edge detection acceleration.

FPGA-Based CNN Inference

Low-resource image classifier (TinyML) on grayscale input (MNIST style).

Ethernet Packet Sniffer

Interface with PHY, decode packet headers, and count traffic types.

Smart Oscilloscope Frontend

Input ADC stream, trigger detection, basic waveform rendering to display or HDMI.

Software and Hardware

Software

Access free tools to design and program FPGAs:

  • Vivado Design Suite (Free WebPACK): HDL design, simulation, and synthesis.
  • Vitis Platform: C/C++ and HLS for Zynq SoCs.
  • PYNQ Framework: Python-based development with Jupyter Notebooks.
  • PetaLinux Tools: Embedded Linux for Xilinx FPGAs/SoCs.
  • QuestaSim (Free Student Edition): Advanced HDL simulation.
  • MATLAB and Simulink with HDL Coder: Model-based design and HDL generation.
  • Verilator and Yosys: Open-source HDL simulation and synthesis.

Supported by AMD University Program tutorials and pynq.io resources.

Hardware

Essential test and measurement equipment for FPGA development:

  • Oscilloscopes: For viewing signal integrity, timing, and digital waveforms.
  • Logic Analyzers: For capturing and analyzing multiple digital signals simultaneously.
  • Signal Generators (Arbitrary Waveform Generators): For creating test signals and stimuli.
  • Spectrum Analyzers: For analyzing frequency components of signals.
  • Digital Multimeters (DMMs): For measuring voltage, current, and resistance.
  • Bench Power Supplies: For providing stable and adjustable DC power to circuits.
  • JTAG Debuggers/Programmers: For in-circuit debugging and programming FPGAs.
  • Soldering Stations & Rework Tools: For assembling and modifying prototype boards.
  • 3D Printers: For creating custom enclosures, mounts, or mechanical parts for projects.
  • Anti-static Mats and Wrist Straps: For protecting sensitive electronic components.
